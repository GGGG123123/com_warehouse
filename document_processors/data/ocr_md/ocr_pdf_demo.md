## 运维代码片段示例

OCR 后的技术文档可能包含代码块。代码块会被单独切成 chunk，避免和普通说明混在一起。

```python
def check_cpu(cpu_percent: float) -> str:
    """根据 CPU 使用率返回告警等级。"""
    if cpu_percent >= 90:
        return "critical"
    if cpu_percent >= 80:
        return "warning"
    return "normal"
```

## 处理流程

1. 先确认 OCR 结果是否完整。
2. 再检查 Markdown 标题、表格、代码块是否正常。
3. 运行切分脚本生成 JSONL。
4. 运行入库脚本写入 Milvus。
5. 通过智能 OnCall 助手提问验证召回效果。
